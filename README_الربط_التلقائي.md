# الإصدار 4.1 — الربط التلقائي مع فرص واستطلاع

## طريقة الرفع على GitHub
1. فك ضغط الملف.
2. ارفع **جميع محتويات المجلد** إلى جذر المستودع نفسه، بما فيها المجلد المخفي `.github`.
3. من GitHub افتح: Settings → Actions → General.
4. تحت Workflow permissions اختر: **Read and write permissions** ثم احفظ.
5. افتح تبويب Actions.
6. اختر الإجراء: **تحديث فرص واستطلاع تلقائياً**.
7. اضغط Run workflow لتشغيل أول تحديث فوراً.
8. بعد انتهاء التشغيل انتظر دقيقة ثم افتح موقع المنصة وحدّث الصفحة.

## التشغيل التلقائي
يعمل الإجراء مرتين يومياً ويحدّث:
`data/opportunities.json`

## ما تم تنفيذه
- جلب آلي للصفحات العامة من فرص واستطلاع بواسطة متصفح Chromium داخل GitHub Actions.
- فلترة مرتبطة بالرياضة، الأندية، الاستثمار، المواقع التجارية، التراخيص والسلامة.
- تنزيل روابط الصور متى كانت ظاهرة في المصدر.
- منع التكرار.
- الاحتفاظ بالبيانات السابقة إذا تعذر أحد المصدرين مؤقتاً.
- مركز مستقل داخل المنصة مع البحث والتصفية والحفظ والانتقال للمصدر.
- زر إضافة المشروع كمرشح لمتابعة المشاريع.

## ملاحظة مهمة
المزامنة تعتمد على الصفحات العامة وليست API رسمية معلنة. إذا غيّرت إحدى المنصتين بنيتها أو منعت القراءة الآلية فقد يحتاج ملف:
`scripts/sync_government_platforms.py`
إلى تحديث.
