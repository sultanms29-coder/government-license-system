منصة العلاقات العامة والحكومية — الإصدار 24.0.0

أبرز التعديلات:
1) إضافة زر صغير «تحديث ومزامنة النظام» في صفحة تسجيل الدخول.
2) الزر يفحص version.json، ويحذف ذاكرة ملفات التطبيق القديمة فقط، ويحدّث Service Worker ثم يعيد تشغيل المنصة.
3) لا يحذف localStorage أو sessionStorage، وبالتالي لا يمس بيانات المستخدمين المحلية أو بيانات المزامنة.
4) التحديثات المركزية تعمل عند رفع النسخة الجديدة إلى نفس رابط الاستضافة؛ يضغط الموظف زر التحديث مرة واحدة.
5) إضافة صورة فارس الحربي إلى الهيكل الإداري.
6) تحديث رقم الإصدار إلى 24.

طريقة نشر تحديث مستقبلي:
- ارفع ملفات الإصدار الجديد إلى نفس مسار الموقع.
- غيّر version داخل version.json واسم CACHE داخل service-worker.js.
- يفتح الموظف صفحة الدخول ويضغط أيقونة التحديث.

تنبيه:
زر التحديث يحدّث ملفات البرنامج، أما مزامنة بيانات الأعمال فتستمر عبر قاعدة البيانات المركزية (Supabase أو خادم الشركة).
