# متابعة التراخيص الحكومية — تعليمات الرفع من الآيفون

ارفع الملفات التالية إلى جذر مستودع GitHub:
- index.html
- manifest.webmanifest
- service-worker.js
- مجلد icons كاملًا

بعد الرفع:
1. انتظر من 1 إلى 3 دقائق.
2. افتح رابط GitHub Pages في Safari.
3. حدّث الصفحة.
4. اضغط مشاركة ← إضافة إلى الشاشة الرئيسية.
5. الاسم المقترح: متابعة التراخيص الحكومية.

إذا ظهرت الأيقونة القديمة:
- احذف الاختصار القديم من الشاشة الرئيسية.
- افتح الرابط مرة أخرى من Safari.
- أضفه من جديد.
