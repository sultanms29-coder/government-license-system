# قائمة النشر الآمن

## قبل الإنتاج
- [ ] اعتماد مالك النظام ومالك البيانات ومسؤول الأمن.
- [ ] جرد البيانات وتصنيفها وتحديد البيانات الشخصية.
- [ ] إعداد إشعار خصوصية وسياسة احتفاظ وإتلاف.
- [ ] تفعيل HTTPS بشهادة موثوقة ومنع HTTP.
- [ ] وضع WAF وRate Limiting أمام المنصة.
- [ ] تفعيل MFA للحسابات الإدارية وجميع المستخدمين عند الإمكان.
- [ ] تفعيل RLS على جميع الجداول واختبارها بحسابات مختلفة.
- [ ] عدم وضع Service Role Key أو كلمات مرور داخل ملفات الواجهة.
- [ ] تقييد CORS بالنطاق الرسمي فقط.
- [ ] جعل Storage خاصًا واستخدام Signed URLs قصيرة المدة.
- [ ] تفعيل سجل تدقيق مركزي وإرسال التنبيهات إلى SIEM.
- [ ] نسخ احتياطي يومي مشفر + نسخة خارجية + اختبار استعادة ربع سنوي.
- [ ] فحص SAST وDependency Scan وDAST واختبار اختراق قبل الإطلاق.
- [ ] خطة استجابة للحوادث وآلية إبلاغ عن خرق البيانات.
- [ ] مراجعة المتطلبات النظامية بواسطة مختص قانوني وأمن سيبراني معتمد.
